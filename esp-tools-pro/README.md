# ESP Flasher Pro

A polished firmware flashing frontend for Espressif devices.

## Features

- Clean professional interface
- Dynamic image rows with offset and file selection
- Robust validation for offset format, duplicate addresses, and file loading
- Status badges for each file input
- Responsive layout for desktop and mobile

## Usage

Open `esp-tools-pro/index.html` in a browser to use the app.

> This is a frontend demo app. For real device flashing, integrate this UI with your existing ESP flashing backend.
